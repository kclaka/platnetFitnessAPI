# PlanetFitness API

## Entity

- Customers
   - Get all customers  (GET)
         `https://planetfitapi.azurewebsites.net/api/customers` 
    - Get one customer by ID (GET)
         `https://planetfitapi.azurewebsites.net/api/customers/:id`
    - Update customer (PUT)
         `https://planetfitapi.azurewebsites.net/api/customers/:id`
    - Delete Customer (DELETE)  
        `https://planetfitapi.azurewebsites.net/api/customers/:id`
    - Create Customer  (POST)
        `https://planetfitapi.azurewebsites.net/api/customers`
